Getting Started

Unzip the archive and find out the folder charityfund-client-files. There are all the template files in this folder.

The file and folder structure is something like this:

  1. /api — folder twitter feed api files.
  2. /css — folder with css files.
  3. /fonts — folder with icon fonts.
  4. /images — folder with image files.
  5. /js — folder with Javascript files.
  6. /php — folder with contact form handler in php code.
  7. index-*.html, ... — Multi and single Main page demos.
  8. News-*.html, ... — News page demos.
  9. page-*.html, ... — Different inner page demos.
  10. shortcode-*.html, ... — Different element page demos.

Upload the template files to server with the help of one of the FTP-clients like FileZilla.

HTML Structure

Divider

1. Basic Divider

Just place a class "divider" to a section to make it divider as like below

<section id="sectionID_if_Needed" class="divider">
  <div class="container">
    <div class="row">
      <div class="col-md-12">
         ------- 
         Content here 
         -------
      </div>
    </div>
  </div>
</section>
2. Fullscreen Divider as like Home section

Just place a class "fullscreen" beside "divider" to make it fullscreen divider

<section id="sectionID_if_Needed" class="divider fullscreen">
  <div class="container">
    <div class="row">
      <div class="col-md-12">
         ------- 
         Content here 
         -------
      </div>
    </div>
  </div>
</section>
3. Divider with Image Background

Just place a class "bg1" beside "divider" to place an image in the background of the divider

<section id="sectionID_if_Needed" class="divider bg1">
  <div class="container">
    <div class="row">
      <div class="col-md-12">
         ------- 
         Content here 
         -------
      </div>
    </div>
  </div>
</section>

There is predefined background classe bg1 in the main-style.css file

.bg1 {
  background-image: url(http://placehold.it/1920x1280);
}

You can also add background image or color by using some predefined attributes like "data-bg-color" and "data-bg-img"

<section data-bg-color="#1A1A1A">
  <div class="container">
    <div class="row">
      <div class="col-md-12">
         ------- 
         Content here 
         -------
      </div>
    </div>
  </div>
</section>

and

<section data-bg-img="images/photos/about-bg1.jpg">

</section>

Fonts used

By default, the template loads Playfair Display, Open Sans and Ubuntu font from Google Web Font Services, you can change the font with the one that suits you best.

You will find the font code in the "main-style.css" file in the "css" folder:

@import url(https://fonts.googleapis.com/css?family=Open+Sans:400,300,600,700,800);
@import url(http://fonts.googleapis.com/css?family=Playfair+Display:400,400italic,700,700italic);
@import url(https://fonts.googleapis.com/css?family=Raleway:400,300,200,500,700,600,800);

Contact Form

Contact form uses php to send emails. So just open contact-form.php file from the php folder of the theme. Change the current email spam@thememascot.com to your email address. Your form is ready.

(NOTE: Contact form requires web server to send emails. Emails can go to your spam/junk folder.)

Google Map

The Map Default code looks like this:

<div class="row">
  <div class="col-md-12 p-0">
    <div class="container-fluid p-0">
      <div class="map-canvas autoload-map" data-height="200" data-address="121 King Street, Melbourne Victoria 3000 Australia" data-zoom="14" data-mapstyle="default" data-marker="images/map-icon-blue.png">
      </div>
    </div>
  </div>
</div>

The Map Defferent Styles (1-9) code looks like this:

<div class="row">
  <div class="col-md-12 p-0">
    <div class="container-fluid p-0">
      <div class="map-canvas autoload-map" data-height="200" data-address="121 King Street, Melbourne Victoria 3000 Australia" data-zoom="14" data-mapstyle="style1" data-marker="images/map-icon-blue.png">
      </div>
    </div>
  </div>
</div>

Replace the value of attribute data-address by your own location & data-mapstyle by change map styles

Flickr Feed

To add Flickr feed, just place the following code:

<div id="flickr-feed" class="clearfix"> 
  <!-- Flickr Link --> 
  <script type="text/javascript" src="http://www.flickr.com/badge_code_v2.gne?count=8&amp;display=latest&amp;size=s&amp;layout=x&amp;source=user&amp;user=52617155@N08"></script> 
</div>

Replace the value of attribute count=8, user=52617155@N08
